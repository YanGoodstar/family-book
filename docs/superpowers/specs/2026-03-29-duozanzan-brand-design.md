# 朵攒攒品牌设计说明

## 背景

当前小程序已经从“预算管理”逐步转向“梦想目标 + 积累进度”的方向，产品核心不再是强调控制开销或复杂财务分析，而是帮助用户看见自己正在一点点靠近想要的生活。

在这个背景下，品牌名称、头像图形和对外文案需要统一到同一套叙事里：

- 不强调金融化、收益感或工具感
- 强调长期积累、温和陪伴和成长感
- 让“存钱”成为实现梦想的过程，而不是冰冷的数字行为

用户已经确认小程序名称为 `朵攒攒`，并认可围绕该名称建立新的品牌表达。

## 目标

本次设计说明要解决四件事：

1. 明确 `朵攒攒` 的品牌含义和产品定位
2. 明确头像图形的设计方向，避免后续视觉实现跑偏
3. 明确小程序的核心语气和主文案
4. 为后续头像重绘、登录页文案、品牌延展提供统一依据

## 设计结论

### 1. 品牌名称

- 小程序名称：`朵攒攒`
- 保留“多攒攒”的谐音记忆点，但不把它作为唯一解释

推荐的对外解释为：

> 朵攒攒，寓意把一点点积累，慢慢攒成会开花的未来。

这个定义同时承接了三层含义：

- `朵`：花苞、成长、期待、未来
- `攒攒`：一点点积累、逐步靠近目标
- 整体产品价值：让用户看见梦想在积累中逐渐成形

### 2. 品牌定位

推荐的产品定位为：

> 一个帮助用户为梦想慢慢积累的小程序。

推荐的品牌核心表达为：

> 把每一次小积累，慢慢攒成一朵花。

这意味着产品要优先传达以下感受：

- 我不是在被迫记账，而是在为未来积累
- 每一笔小储蓄都算数
- 这个产品是在温和陪伴我，而不是催促我

### 3. 品牌气质

整体品牌气质统一为：

- 简洁
- 高级
- 温和
- 克制
- 有成长感

需要主动避免的方向：

- 过强的金融感
- 明显的招财感
- 过度工具化的记账产品气质
- 过于卡通、幼态或花哨的视觉表达

## 头像设计方向

### 1. 最终方向

头像图形方向确定为：

> 几何花苞 + 隐藏元宝负形

推荐采用“第一眼看到花苞，第二眼看出攒钱寓意”的双层识别方式。

### 2. 图形原则

头像图形应满足以下原则：

- 第一识别层必须是“将开未开的花苞”
- 第二识别层通过中心留白或内轮廓，隐约形成“元宝”形态
- 元宝只能作为隐藏含义，不能画得过于直白或写实
- 图形整体采用圆润的几何结构，不使用尖锐折角
- 图形主构图偏竖向，强化“向上生长”的感觉

### 3. 构图原则

推荐头像采用纯图形徽章，不加文字。

构图建议：

- 使用圆角方形底或完整徽章底形
- 中央放置单一主图形，不叠加房子、账本、金币等多重元素
- 图形占比控制在可读范围内，确保小程序头像尺寸下仍能辨认
- 保持足够留白，避免缩小时显得拥挤

### 4. 为什么不采用其他方向

本次讨论过的其他方向包括：

- 花朵 + 账本
- 花芯生长符号
- 花苞 + 更强的元宝显性表达

最终不作为主方向的原因如下：

- `花朵 + 账本`：业务含义更直白，但容易落回普通记账工具图标
- `花芯生长符号`：品牌感更强，但“攒”的含义偏弱
- `强化元宝表达`：容易显俗，且会削弱“成长感”主叙事

因此最终选择“花苞为主、元宝为隐喻”的方案。

## 色彩方向

### 1. 主色系统

主色建议采用“生长感绿色”，而不是典型金融绿。

色彩结构建议：

- 主色：自然、清透的绿色
- 深色：更稳重的深绿，用于层次和对比
- 中和色：接近奶白或雾白的浅色，用于提亮和留白

### 2. 金色使用原则

金色不作为主色使用。

如需使用，仅允许作为极弱点缀，用来暗示“积累的价值感”，但不能抢走绿色主基调。否则会让品牌气质从“慢慢开花”偏向“招财”和“功利”。

## 文案体系

### 1. 品牌主句

推荐品牌主 slogan：

> 把每一次小积累，慢慢攒成一朵花。

这是最完整的一句品牌表达，适合用于：

- 欢迎页主文案
- 品牌说明页
- 视觉稿主标题
- 对外介绍中的核心一句话

### 2. 辅助文案

推荐辅助文案：

> 今天攒一点，未来开一朵。

适合用于：

- 引导弹层
- 空状态文案
- 登录后欢迎提示
- 梦想目标模块轻提示

### 3. 产品化文案

推荐功能型文案：

> 把想要的生活，慢慢攒出来。

适合用于：

- 说明页
- 功能介绍
- 个人中心介绍语

### 4. 小程序简介

推荐作为微信公众平台或小程序简介使用的版本：

> 朵攒攒是一款帮助你为梦想慢慢积累的小程序，记录每一次存下的钱，看见目标一点点开花。

如果需要更短版简介，可使用：

> 为梦想攒一点，也为生活多一点从容。

## 设计边界

本次设计说明只定义品牌方向，不直接规定以下具体产物：

- 最终头像 SVG 路径细节
- 完整品牌视觉规范手册
- 全站 UI 改版方案
- 插画系统或默认头像系统

这些内容应在后续实现或视觉延展阶段，基于本说明继续展开。

## 对现有产品的影响

基于本次设计结论，后续相关页面和素材应逐步统一：

- 小程序头像应朝“几何花苞 + 隐藏元宝负形”重绘
- 登录页、授权页、欢迎语应统一到“积累会开花”的叙事
- 梦想目标相关空状态、完成状态、提醒文案应优先使用温和陪伴式表达
- 避免在新页面里重新引入过强的金融化图标和措辞

## 验收标准

当后续视觉实现完成时，应满足以下验收标准：

1. 用户看到名称 `朵攒攒` 时，可以自然联想到“积累”和“成长”
2. 头像第一眼像花苞，细看能感受到积累寓意，但不会显得俗气
3. 文案整体呈现为温和、克制、长期陪伴，而非理财营销话术
4. 品牌表达与当前“梦想目标 / 存钱进度”产品方向保持一致
5. 后续页面扩展时，有明确的名称、图形和文案基准可遵循

## 后续建议

建议按以下顺序继续推进：

1. 基于本说明重绘一版更贴合 `朵攒攒` 的主头像
2. 将欢迎语、登录后提示、空状态文案统一到品牌主句和辅助文案体系
3. 视需要补充一页轻量级品牌使用说明，供后续页面设计保持统一
