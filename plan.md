# 家庭记账小程序 - 需求计划文档

## 1. 项目概述

开发一款面向家庭用户的记账小程序，帮助用户记录日常收支、管理家庭财务、分析消费习惯，并通过梦想目标管理实现强制储蓄。

**目标用户：**
- 家庭主妇/主夫
- 年轻上班族
- 需要管理家庭财务的用户

---

## 2. 功能需求

### 2.1 记账功能
- [ ] **快速记账**: 一键记录收支，支持金额、分类、备注、时间
- [ ] **分类管理**: 预设常见收支分类，支持自定义分类
- [ ] **记账提醒**: 每日定时提醒记账
- [ ] **语音记账**: 语音输入快速记账
- [ ] **拍照记账**: 拍照识别发票、小票自动记账

### 2.2 账单管理
- [ ] **账单列表**: 按时间倒序展示所有账单
- [ ] **账单筛选**: 按时间、分类、金额筛选
- [ ] **账单编辑**: 修改、删除已记录账单
- [ ] **账单搜索**: 关键词搜索历史账单

### 2.3 数据统计
- [ ] **收支概览**: 月度/年度收支总览
- [ ] **分类统计**: 饼图展示各类别支出占比
- [ ] **趋势分析**: 折线图展示收支趋势
- [ ] **预算分析**: 对比实际支出与预算

### 2.4 用户功能
- [ ] **微信登录**: 一键授权登录
- [ ] **个人中心**: 用户信息管理
- [ ] **数据同步**: 多设备数据同步
- [ ] **数据备份**: 云端数据备份与恢复

### 2.5 家庭共享
- [ ] **家庭组**: 创建/加入家庭组
- [ ] **成员管理**: 添加、移除家庭成员
- [ ] **共享账本**: 家庭成员共同记账
- [ ] **权限设置**: 管理员、普通成员权限区分

### 2.6 预算管理
- [ ] **月度预算**: 设置每月总预算
- [ ] **分类预算**: 为不同类别设置预算
- [ ] **预算预警**: 超预算时提醒
- [ ] **预算报表**: 预算执行情况分析

### 2.7 资产管理
- [ ] **账户管理**: 现金、银行卡、支付宝、微信等账户
- [ ] **资产统计**: 总资产、净资产统计
- [ ] **转账记录**: 账户间转账记录

### 2.8 账单导出
- [ ] **Excel导出**: 导出账单为Excel文件
- [ ] **数据分享**: 分享账单统计图片

### 2.9 梦想目标管理

#### 2.9.1 目标设定
- [ ] **创建目标**: 设定目标名称（如买车、旅游）、目标金额、目标期限
- [ ] **储蓄计划**: 支持固定金额储蓄（如每月存5000元）或按工资百分比储蓄（如每月存工资的30%）
- [ ] **多目标管理**: 同时管理多个储蓄目标

#### 2.9.2 储蓄跟踪
- [ ] **自动累加**: 每月根据设定金额自动累加"已存金额"
- [ ] **进度展示**: 实时显示目标完成进度（百分比 + 进度条）
- [ ] **完成预测**: 自动计算预计完成时间（如"按当前进度，还需9个月完成"）

#### 2.9.3 目标看板
- [ ] **当前目标进度**: 已存金额 / 目标总额
- [ ] **预计剩余时间**: 显示还需月数
- [ ] **储蓄对比**: 本月应存金额 vs 实际可存金额
- [ ] **趋势图表**: 近6个月储蓄趋势图

#### 2.9.4 预警提醒
- [ ] **储蓄预警**: 当某月实际支出超出预算，导致无法存够目标金额时自动提醒
- [ ] **弹窗通知**: 首页弹窗提示"本月支出过高，目标储蓄未达标，建议控制开销"
- [ ] **消息推送**: 储蓄进度更新、目标即将达成等通知

## 4. 注意事项

1. **数据安全**: 用户财务数据敏感，需做好加密和权限控制
2. **性能优化**: 账单数据量大时，需做好分页和懒加载
3. **用户体验**: 记账流程要简洁，三步内完成记账
4. **兼容性**: 适配不同尺寸的手机屏幕
5. **合规性**: 遵循微信小程序平台规范
